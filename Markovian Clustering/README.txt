Software Requirements - 
Python 2.7 
It requires numpy and networkx packages
-- pip install from commandline or get numpy from http://docs.scipy.org/doc/numpy-1.10.1/user/install.html 

The code operates on 4 files of a family namely:
A.fo
A.net
A-B.sim
B.fo
B.net 

If you want to test with any of the network families, please copy these files 
into the folder location of source file(python_mcl-master.py) and run python_mcl-master.py.
